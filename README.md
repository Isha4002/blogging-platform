# 🚀 BlogHub - MERN Blogging Platform

A modern full-stack blogging platform built using the MERN stack. Users can register, log in, create blogs with a rich text editor, upload cover images, like blogs, comment, bookmark posts, and manage their own content.

## 🌐 Live Demo

**Frontend:** https://blogging-platform-tau-sand.vercel.app

**Backend API:** https://blogging-platform-zbnp.onrender.com

---

## ✨ Features

- 🔐 JWT Authentication
- ✍️ Rich Text Editor (Tiptap)
- 🖼️ Cloudinary Image Upload
- 📝 Create, Edit & Delete Blogs
- ❤️ Like Blogs
- 💬 Comment System
- 🔖 Bookmark Blogs
- 📈 Trending Blogs
- 🔍 Search Blogs
- 📄 Pagination
- 👤 User Dashboard
- 📚 My Blogs
- 📱 Responsive UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Isha4002/blogging-platform.git
```

### Install Dependencies

Backend

```bash
cd server
npm install
```

Frontend

```bash
cd client
npm install
```

---

## ▶️ Run Locally

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
